The Browser tool allows users to search for {{page.atoms}}. When opening the browser page (Figure 3), there are two options:
1. scroll the full list of {{page.atoms}} and select one to see more detailed information (Browse all);
2. type the {{page.atom}} name or abbreviation in the Search bar.

![Browser Page]({{site.figures_link}}/{{page.portal}}/browse.png)
{: .text-center }

_Figure 3: Browser page_
{: .text-center }
In both cases, the search can be further refined by using the filters provided on the left side of the page (Show semantic resource views, Show retired semantic resources, Categories, Groups, Natural languages, Formality levels, Semantic resource types). Upon selecting the filter, the search will be updated to show results which contain the selected filter. Repeat the process to add additional filters.

Moreover, using the tabs on the right side of the search bar, the list of {{page.atoms}} can be sorted by choosing the type of format (SKOS, OWL, UMLS, OBO) or other desired criteria:
