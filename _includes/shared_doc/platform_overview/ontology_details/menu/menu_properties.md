### Properties
In the summary page of the chosen {{ page.atom }}, by clicking on the Properties tab all the properties used within the selected {{ page.atom }} are displayed (Figure 20). Similar to the Classes/Concepts tab, the tree-like structure is shown on the left, while the property details are shown on the right.
