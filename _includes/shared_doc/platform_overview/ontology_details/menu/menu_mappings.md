### Mappings
The Mappings tab displays the mappings computed by {{page.portal}} for the selected resource, grouped by {{ page.atom }} (Figure 22). The page includes a table with two columns: “Semantic artefact” and “Mappings” and a bubble plot. The first column identifies the {{ page.atom }} to which the selected resource is mapped, and the second column shows the number of mappings. By clicking on a {{ page.atom }} all the mappings are shown (Figure 23).
